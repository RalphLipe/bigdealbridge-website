# Big Deal Bridge — Website

Marketing site for [Big Deal Bridge](https://bigdealbridge.com), the duplicate bridge app for iPhone and iPad.

Hosted via GitHub Pages at https://bigdealbridge.com

## Structure

```

/faq/               → faq/index.html    (FAQ)
/support/           → support/index.html (support)
/privacy/           → privacy/index.html (privacy policy)
```
